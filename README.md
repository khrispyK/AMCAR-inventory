# AMCAR-inventory-
The AMCAR Inventory is a local web-based barcode scanning system.
Phones will scan barcodes → send data to laptop → store results in CSV.

This MVP covers:
✔ Login
✔ Barcode scanning
✔ Auto-fetch part description from CSV
✔ User input for quantity & location
✔ Append scanned data to a second CSV
✔ Option to scan again or exit
